# **App Name**: AI for Everyone - Kerala

## Core Features:

- Hero Section: Displays the main title, subtitle, CTAs (Get Started, Learn More), key stats (1L+ Target Learners, 100% Community Focus, 3 Learning Pillars), and a 'Powered by TinkerHub' badge.
- Why Kerala Section: Explains the initiative's context within Kerala's digital literacy legacy, emphasizing its role as a global model, enabler of creators, and promoter of knowledge as a public good.
- Learner Segments Section: Showcases the three learner groups (KuttyMakers, Young Makers, Friends of the Movement) with detailed cards, age badges, accent colors, clear layout, and glassmorphism effects.
- Three Pillars Section: Highlights the learning approach - Head, Hand, Heart - using distinct gradient cards, icons, titles, subtitles, and descriptions.
- Challenges Section: Explains the need for AI literacy by presenting challenges related to skills gap, workforce impact, and misinformation, including a special callout box about protecting communities from deepfakes.
- Partnership Section: Enables users to learn about becoming knowledge, financial, media & outreach, resource, and community partners
- Final Quote Display: A section displaying an impactful, centered quote about Kerala's role in the AI revolution, designed with large, elegant typography, gradient text, a subtle background glow, and refined attribution text.

## Style Guidelines:

- Background: Deep slate/purple dark background (#1E1E2A) to create a modern, premium feel.
- Primary: Cyan (#00FFFF) as a vibrant accent color to highlight key elements and CTAs, embodying a forward-thinking and innovative vibe. Avoid Teal!
- Accent: Pink (#FF69B4) as a secondary accent color to create visual interest and depth, complementing the cyan and purple tones.
- Headline font: 'Poppins', a geometric sans-serif to convey a precise, contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif that balances a modern look with warmth and personality.
- Lucide-react icons with subtle micro-animations on hover (pulse, float, rotate) to add interactivity.
- Glassmorphism effects with frosted glass/backdrop blur to create depth. Use floating gradient orbs in the background.
- Smooth animations and hover effects on all interactive elements to enhance user experience.
